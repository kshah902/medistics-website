# Medistics Website

Modern Care Management for Independent Medical Practices and Pharmacies.

## About

Medistics is a care management technology and services company built for practices that want predictable revenue, better patient outcomes, and zero operational chaos.

## Features

- **Liquid Glass Design** - Modern glassmorphism UI with blue theme
- **Fully Responsive** - Works on all devices
- **Animated Elements** - Smooth scroll animations and interactive effects
- **Contact Form** - Built-in form with validation

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)

## Getting Started

Simply open `index.html` in a web browser or serve the directory with any static file server.

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve
```

## Structure

```
medistics-website/
```
